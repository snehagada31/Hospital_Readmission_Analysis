# Hospital_Readmission_Analysis

## Overview

This project analyzes hospital readmission data using Python. The goal is to understand patterns in patient readmissions through data cleaning, exploratory data analysis, and visualization.

## Tools Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Project Files

* `Hospital_Readmission_Analysis.ipynb` – Main notebook containing the analysis.
* `hospital_readmissions.csv` – Dataset used for the project.
* `README.md` – Project documentation.

## Steps Performed

* Loaded the dataset
* Cleaned and prepared the data
* Explored the data using Pandas
* Created visualizations using Matplotlib and Seaborn
* Derived insights from the analysis

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/snehagada31/Hospital_Readmission_Analysis.git
```

2. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open the notebook:

```bash
jupyter notebook Hospital_Readmission_Analysis.ipynb
```
